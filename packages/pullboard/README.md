# pullboard

The short alias for the Pullboard CLI — `npx pullboard` runs the exact same tool as [`@pullboard/cli`](https://www.npmjs.com/package/@pullboard/cli), just with a name that's quicker to type and say.

```sh
npx pullboard init                    # provision a workspace + token, no signup
npx pullboard build item-123 -- npm test   # claim → build → submit with real git evidence
```

See [`@pullboard/cli`](https://www.npmjs.com/package/@pullboard/cli) for the full command reference. Part of [pullboard-node](https://github.com/pullboard-dev/pullboard-node) — the official Node SDK + CLI for [Pullboard](https://pullboard.dev).
